VOUS AVEZ ÉTÉ VICTIME D'HAMEÇONNAGE !

Si vous êtes arrivé(e) sur cette page, ce n'est pas une erreur : vous êtes tombé(e) dans notre piège !

Rassurez-vous, vos données sont en sécurité.

Mais si ceci avait été une véritable attaque, un logiciel malveillant pourrait être en train de chiffrer vos données et les exposer publiquement en ce moment même.

Pourquoi êtes-vous ici ?

Vous avez cliqué sur un lien piégé ou scanné un QR code suspect !

3 réflexes pour identifier les menaces

  • Vérifiez l'expéditeur : L'adresse de l'envoyeur semble-t-elle légitime ?
  • Analysez le lien : Placez votre curseur sur le lien pour voir la destination réelle avant de cliquer.
  • Doutez de l'urgence : Un message qui exige une action immédiate est souvent un piège !

Nos indices de piège

  • Le texte était volontairement bourré de fautes d'orthographe.
  • Le lien renvoyait vers une page étrange (ne se terminant pas par .fr).
  • Le QR Code était une étiquette collée par-dessus un support existant.
