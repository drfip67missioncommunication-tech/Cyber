Si vous êtes arrivé(e) sur cette page, ce n'est pas une erreur : vous êtes tombé(e) dans notre piège !

Rassurez-vous, vos données sont en sécurité.

Mais si ceci avait été une véritable attaque, un logiciel malveillant pourrait être en train de chiffrer vos données et les exposer publiquement en ce moment même.

Pourquoi êtes-vous ici ?

Vous avez cliqué sur un lien piégé ou scanné un QR code suspect !

3 réflexes pour identifier des menaces

  • Vérifiez l'expéditeur : Connaissez-vous la personne qui vous a écrit ?
  • Analysez le lien : Placez votre curseur sur le lien, et regardez vers quelle page il vous dirigera (en bas à gauche de votre fenêtre).
  • Doutez de l'urgence : Un message qui demande une action immédiate est souvent un piège !

Nos pièges

Nous n'allons pas vous laisser partir sans vous dire ce que nous avons fait !

  • Le texte était bourré de fautes d'orthographe.
  • Le lien renvoyait vers une page étrange (ne se terminant pas par .fr).
  • Le QR Code de l'affiche était une étiquette collée sur un autre QR Code !
